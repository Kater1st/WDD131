Basketball Association Website - Site Plan

Site Name

Mashcentral Basketball Association

The name was chosen to reflect the central role of the site in providing essential information and connection for local basketball enthusiasts. It communicates clearly the website's purpose as a hub for all things related to the basketball association.

Suggested domain name: mcba.org

Site Purpose

The site serves as a centralized platform where users can find up-to-date information about the association, teams, match schedules, events, and membership opportunities. It aims to foster a community around basketball, offering resources for players, coaches, and fans alike.

Color Schema

Primary Text Color: white - Used for headings, buttons, and highlights

Secondary Text Color: black - Used for headings, buttons, and highlights on bright background

Primary Color: #ff3f57 - Used for headings, buttons, and highlights

Secondary Color: 1e1e1e (Dark Gray) - Used for footers and background elements to provide a contrast and professional look.

Typography

Font Selection 1: Montserrat for headings, giving a modern and clean look. Applied to all main titles and section headers.

Font Selection 2: Roboto for body text, offering readability and clarity throughout the site.

Wireframe

Mobile View: The layout will feature a collapsible navigation menu, a hero section with a brief introduction, followed by sections for teams, events, and contact info.

Mobile view wireframe sketch

Desktop View: The homepage will have a horizontal navigation bar with dropdowns, a large banner for association news, and grid-based content sections for easy access to information about teams, events, and membership.

Desktop view wireframe sketch

Wireframe Sketch (Desktop View)

Header: Logo and navigation bar

Main Section: Introduction, Teams & Schedules, Events

Footer: Contact Information, Social Media Links

Wireframe Sketch (Mobile View)

Header: Logo and collapsible navigation menu

Main Section: Scrolling content with team lists, upcoming events, and a contact form

Footer: Links to membership and social media

Testing

To ensure optimal user experience, the website will be tested using validation tools for HTML, CSS, and accessibility. I will also ensure compatibility across different browsers and devices using performance and SEO best practices.
